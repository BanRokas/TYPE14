Tag'ų sąrašas

  • h1-h6 - Heading - Antraštė skirta rašyti pavadinimams
  • p - Paragraph - Paragrafas
  • img - Image - Paveiksliukas
  • a - Hypertext Anchor (hyperlink) - Nuoroda
  • br - Line Break - Eilutės lūžis
  • hr - Thematic Break (horizontal line) - Eilutės lūžis su brūkšniu
  • ol - Ordered list - Rikiuotas sąrašas
  • ul - Unordered list - Nerikiuotas sąrašas
  • li - List item - Sąrašo elementas
  • dl - Description list (detailed list) - Aprašomasis (detalusis) sąrašas
  • dt - Description term - Aprašomojo sąrašo terminas
  • dd - Description definition - Termino aprašymas
  • div - Tuščias blokinis elementas skirtas grupuoti kitus elementus
  • link - Skirtas prijungti kitus failus (pagrinde CSS) prie kodo
  • script - Skirtas rašyti/prijungti JS kodą
  • style - Skirtas rašyti CSS kodą
  • table - Lentelė
  • caption - Lentelės pavadinimas
  • colgroup - column group - Lentelių grupės
  • col - column - Stulpelis
  • thead - table head - Lentelės viršutinė dalis
  • tbody - table body - Lentelės kūnas
  • tfoot - table foot - Lentelės apatinė dalis
  • tr - table row - Lentelės eilutė
  • th - table heading - Lentelės stulpelio pavadinimas
  • td - table data - Lentelės duomenys
  • form - Forma
  • label - Etiketė prikabinama prie formos įvesties lauko
  • input - Įvesties laukas su LABAI DAUG skirtingų tipų
  • textarea - Įvesties laukas skirtas didesniam kiekiui teksto
  • select - Pasirinkimo laukelis
  • option - Pasirinkimo variantas

Atributų sąrašas

  • lang - language - Nurodo puslapyje vyraujančią kalbą
  • charset - character set - Simbolių rinkinys
  • src - source - Šaltinis nurodo failą
  • href - hyperlink reference - Nurodo nuorodą/failą
  • alt - Alternative text - Alternatyvus tekstas elemento neužkrovimo atvejui
  • target - Nurodo/Nutaiko kur bus atidaryta paspausta nuoroda
  • width - Nurodo elemento plotį
  • height - Nurodo elemento aukštį
  • rel - relation - Nurodo nurodomo failo santykį su esamu failu
  • defer - Nurodo, kad failas būtų perskaitytas po to kai užsikraus puslapis
  • rowspan - Užimamų eilučių kiekis
  • colspan - Column span - Užimamų stulpelių kiekis
  • span - Stulpelių kiekis (naudojamas colgroup viduje esančiam col tag'ui)
  • for - Skirtas nurodyti kuriam įvesties laukui yra skirta atiketė
  • type - Skirtas nurodyti input'o tipą
  • name - Įvesties laukelio vardas (pagal kurį pasiimama informacija)
  • id - Unikalus elemento pavadinimas (į jį nurodo for atributas)
  • class - Elemento klasė (skirta priskirti stilius)
  • cols - textarea įvesties lauko stulpelių kiekis(ilgis)
  • rows - textarea įvesties lauko eilučių kiekis(aukštis)
  • placeholder - Tekstas nusakantis kas turi būti įrašyta į įvesties laukelį
  • value - option tag'o vertė, kuri bus perduodama script'ui
