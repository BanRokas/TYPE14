Box Model

  1. Content - Turinys - Elemento informacija turinti plotį/aukštį
  2. Padding - Pamušalas - Tarpas esantis tarp elemento turinio ir jo rėmelio (matomos elemento kraštinės)
  3. Border - Rėmeliai - Linija tarp elemento pamušalo ir paraštė
  4. Margin - Paraštė - Tarpas esantis tarp elemento rėmelio/krašto iki kito elemento rėmelio/krašto.
  5. Position(t/r/b/l) - Pozicija(v/d/a/k) - Elemento atitraukimas nuo nurodytų kraštinių (viršus/dešinė/apačia/kairė).

Content

Elemento turinys. Jo dydis keičiamas arba dedant daugiau turinio arba keičiant turinio dydį arba paties elemento dydį.
Elemento turinys yra vizualiai matomas vartotojui. Yra spalvinamas fono spalvos.

  • width
  • height
  • font-size

Padding

Tarpas tarp elemento turinio ir rėmelio/kraštinės. Jis yra vizualiai matomas. Jis yra fono dalis.
Galima nustatyti visas kraštines vienu stiliumi, arba kiekvieną kraštinę aprašinėti atskirai.

  • padding: top/right/bottom/left
  • padding: top/bottom right/left
  • padding: top right/left bottom
  • padding: top right bottom left
  • padding-top: top
  • padding-right: right
  • padding-bottom: bottom
  • padding-left: left

Border

Linija tarp elemento pamušalo ir paraštės. Yra vizualiai matomas. Papildo elemento dydį.
Rėmelis turi šiuos nustatymus: plotis, stilius, spalva, kampo apvalinimas. Kiekvienas iš šių stilių gali būti aprašytas atskirai kiekvienai kraštinei/kampui arba bendrai.

  • border: width style color
  • border-width: width
  • border-style: style
  • border-color: color
  • border-radius: %/px/em/rem
  • border-[corner]-radius: %/px/em/rem
  • border-[side]-[width/style/color]: [width/style/color]

Margin

Tarpas tarp elemento rėmelio/kraštinės ir kito elemento rėmelio/kraštinės/margin'o. Jis nėra vizualiai matomas.
Galima nustatyti visas kraštines vienu stiliumi, arba kiekvieną kraštinę aprašinėti atskirai.
Labai patogu naudoti "margin: 0 auto" norint išcentruoti blokinį elementą.

  • margin: top/right/bottom/left
  • margin: top/bottom right/left
  • margin: top right/left bottom
  • margin: top right bottom left
  • margin-top: top
  • margin-right: right
  • margin-bottom: bottom
  • margin-left: left

Positions (top/right/bottom/left)

Suteikus relatyvią poziciją galima nurodyti per kokį atstumą nuo nurodytos kraštinės elementas atsitrauks.
