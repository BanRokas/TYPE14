Paveldėjimas

Iš tėvinio elemento, vaikiniai elementai paveldi tam tikrus stilius. Paveldimi stiliai dažniausiai būna paprasti kaip: color, font-size, font-family... Nepaveldimi stiliai yra: width, height, display, border, fonai...

Svetainių numatytieji stiliai

Svetainių numatytieji stiliai

Tag'ai gali turėti savitus stilius, kurie jiem yra suteikti pačių naršyklių.
Kartais skirtingos naršyklės turi minimalistiškai skirtingus numatytuosius elementų stilius. Tai gali paveikti svetainės išvaizdą ją atidarant per skirtingas naršykles.

Yra keletas būdų kaip spręsti šias problemas.
Vienas iš būdų naudoti Reset CSS, kuris panaikins visų tag'ų savybes / suvienodins visus tag'us (not very good option) Reset CSS.
Kitas būdas yra naudoti Normalize CSS, kuris suvienodina stilius tarp naršyklių ir panaikina labai erzinančius stilius (naudojamas dažniau) Normalize CSS.
