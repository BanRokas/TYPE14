Basic Selectors

  1. element - nurodytas/i elementas/ai (tag'as)
  2. .class - elementas/ai su nurodyta klase
  3. #id - elementas su nurodytu ID
  4. * - viskas
  5. selector selector - vieno selektoriaus viduje betkur esantis kitas selektorius
  6. selector > selector - vieno selektoriaus viduje iškart esantis kitas selektorius
  7. selector, selector - abiem selektoriam nurodomi stiliai vienu metu
  8. selectorSelector - elementas turintis abu/visus nurodytus selektorius
  9. selector1 + selector2 - elementas(selector2) einantis iš karto už kito elemento(selector1)
  10. selector1 ~ selector2 - elementas(selector2) einantis už kito elemento(selector1)

Pseudo Selectors

  1. :link - nelankyta nuoroda
  2. :visited - lankyta nuoroda
  3. :hover - kai pelės kūrsorius užėjęs ant elemento
  4. :active - paspaudimo momentas
  5. :empty - tuščias elementas (be teksto/tarpų/tag'ų)
  6. :fullscreen - elementas esantis pilno ekrano būsenoje
  7. :not(selector) - nurodo, kuris elementas neturi kažkokio stiliaus
  8. :root - parenka dokumento šaknis (ne stilizuoti, o kurti kintamuosius)
  9. :target - per nuorodą parinktas elementas
  10. :first-child - pirmasis vaikinis elementas
  11. :last-child - paskutinis vaikinis elementas
  12. :only-child - vienintelis vaikinis elementas
  13. :first-of-type - pirmasis kažkokio tipo elementas
  14. :last-of-type - paskutinis kažkokio tipo elementas
  15. :only-of-type - vienintelis kažkokio tipo elementas
  16. :nth-child(nr) - kelintas(nr) vaikinis elementas
  17. :nth-last-child(nr) - kelintas(nr) vaikinis elementas nuo galo
  18. :nth-of-type(nr) - kelintas(nr) kažkokio tipo elementas
  19. :nth-last-of-type(nr) - kelintas(nr) kažkokio tipo elementas nuo galo
  20. ::before - prieš turinį esantis turinys
  21. ::after - po turinio esantis turinys
  22. ::marker - sąrašo elementų žymėklis
  23. ::first-letter - pirmoji elemento raidė
  24. ::first-line - pirmoji elemento eilutė (kuri atvaizduojama ekrane)
  25. ::selection - pažymėta informacija
