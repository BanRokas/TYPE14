CommandPrompt / Terminal

  • cd - change directory - Vaikščioti tarp aplankalų
  • dir - list directories - Parodyti visus aplankalus/failus esančius dabartinėje direktorijoje (windows)
  • ls - list directories - Parodyti visus aplankalus/failus esančius dabartinėje direktorijoje (linux/mac)
  • mkdir - make directory - Sukuria naują aplankalą
  • touch - Sukuria naują failą
  • cat - Concatenate - Išveda failo informaciją arba sukuria naują failą ir leidžia į jį kažką rašyti jeigu naudojama su "> naujofailoVardas.tipas"
  • rm -r - Ištrina nurodytą aplankalą/failą rekursyviai(-r)
  • clear - Išvalo terminalo vaizdą
  • CTRL C - Nutraukia vykdomą komandą / Naikina rašomą eilutę
  • ARROW UP/DOWN - Pateikia paskutinę/sekančią vykdytą komandą
  • TAB - užbaigia rašomą pavadinimą / pasiūlo užbaigimus

  • git config --help

Git Commands

  • git init - inicializuoja git'o repozitoriją
  • git status - pateikia git'o repozitorijos statusą
  • git add failas.tipas - add'ina nurodytą/us failą/us į stage'ing area (paruošia failus commit'inimui (kodo versijos sukūrimui))
  • git rm --cached failas.tipas - remove'ina nurodytą/us failą/us iš stage'ing area (išima paruoštus commit'inimui failus iš paruošimo vietos)
  • git restore --staged - remove'ina nurodytą/us failą/us iš stage'ing area (išima paruoštus commit'inimui failus iš paruošimo vietos)
  • git commit -m "žinutė" - Sukuria kodo versiją
  • git checkout versijosID - perkels į nurodytą kodo versiją
  • git remote add origin HTTPS/SSH - prijungia lokalią repozitoriją prie nurodytos išorinės repozitorijos
  • git branch -m main - Pervadina dabartinę šaką į nurodytą pavadinimą
  • git push - Nusiunčia naujausią/as kodo versiją/as į GitHub
  • git pull - Atsiunčia naujausią/as kodo versiją/as iš GitHub
  • git clone HTTPS/SSH - Klonuoja nurodytą išorinę repozitoriją į esamą aplankalą
