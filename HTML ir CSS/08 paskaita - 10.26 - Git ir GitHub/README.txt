Rytmečio metu savarankiškai nuveikti:
1) Peržiūrėti CAO (nieko nedaryti).
2) Atsisiūsti ir įsirašyti git https://git-scm.com/download/win
3) Peržiūrėti git documentation videos https://git-scm.com/videos (TIK žiūrėti)
4) Susikurti GitHub paskyrą.
5) Pasimokinti pradines(basic) commandPrompt/Terminal komandas https://google.lt
6) Pabandyti config'inti git'ą kompiuteryje globaliai.
7) Atlikti CAO užduotį.
8) Pasimokinti git komandas.

extra) Pasidomėti git GUI ir jei randat tinkantį/patinkantį - įsirašyti ir naudoti ateityje.
